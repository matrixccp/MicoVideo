# MicoVideo

#### 项目介绍
基于MVP开发的一款小视频+音乐app

为了解决WebView容易出现崩溃的问题采用WebView独立进程解决方案

双进程间通信使用AIDL进程间通信技术

#### 软件架构
软件架构说明
Kotlin+RxJava2 + RxAndroid2 + Glide +LeakCanarya+Retrofit2+okhttp3+MVP

#### 安装教程

1. app 业务逻辑module
2. multiple-status-view 多状态视图
3. weblib 独立进程webview

#### 使用说明

1. 附件下载apk文件安装体验

#### 鸣谢

感谢weblib  multiple-status-view开源作者
